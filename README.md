Este repositorio está diseñado para aprender algunos trucos prácticos que permiten trabajar más rápido y de forma más eficiente en Visual Studio Code.
Objetivo

Aprender atajos que aceleran el flujo de trabajo.

Configurar correctamente el entorno de desarrollo.

Conocer extensiones clave para programación.

Optimizar la experiencia en VS Code.
Trucos que veremos

Uso eficiente de atajos de teclado.

Edición múltiple de líneas.

Búsqueda y reemplazo avanzado.

Uso del Command Palette.

Navegación rápida entre archivos.

Formateo automático de código.

Ejemplos prácticos.

Atajos explicados.

Configuraciones recomendadas.

Buenas prácticas para organizar tu entorno.
